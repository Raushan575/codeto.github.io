Installing Python

Download Python from python.org and install it. Check installation:

Python Syntax

Comments

Keywords

Keywords are reserved words like if, else, for, while, def, class.

Variables & Data Types

name = "Rahul" # string age = 21 # int price = 99.5 # float is_active = True # boolean

Type Casting

x = "10" y = int(x) print(y + 5)

Input & Output

name = input("Enter name: ") print("Welcome", name)

Indentation

Python uses indentation instead of brackets.

if age > 18: print("Adult")

Operators

Arithmetic: + - * / % //

Assignment: = += -=

Comparison: == != > <

Logical: and or not

Bitwise: & | ^

Membership: in, not in

Identity: is, is not

Control Flow

If, Else, Elif

if marks >= 75: print("Distinction") elif marks >= 40: print("Pass") else: print("Fail")

Nested If

if age > 18: if has_id: print("Allowed")

Match Case (Python 3.10+)

match day: case 1: print("Monday") case 2: print("Tuesday")

Loops

for i in range(1,6): print(i) x = 0 while x < 5: x += 1

Break, Continue, Pass

for i in range(5): if i == 3: break print(i)

Data Structures

Strings

Lists

Tuples

Sets

Dictionaries

List Comprehension

squares = [x*x for x in range(5)]

Dictionary Comprehension

data = {x: x*x for x in range(5)}

Functions

Defining Functions

def add(a, b): return a + b

Function Arguments

def greet(name="User"): print("Hello", name)

*args and **kwargs

def demo(*args, **kwargs): print(args, kwargs)

Lambda Function

square = lambda x: x*x

Recursive Function

def fact(n): return 1 if n==0 else n*fact(n-1)
